## рҹ“Ҡ Monitoring Stack / ШӯШІЩ…Ш© Ш§Щ„Щ…ШұШ§ЩӮШЁШ©

### Prometheus
- **Auto-discovery** of pods via Kubernetes SD
- **Custom metrics** from Flask app (`app_requests_total`, `app_up`, `app_uptime_seconds`)
- **Alert rules**: AppDown, HighRequestCount, PodRestart
- **Retention**: 3 days with 2Gi storage limit

### Grafana
- **Pre-configured datasource** (Prometheus вҖ” auto-provisioned)
- **Custom dashboard** with 6 panels:
  - App Status (UP/DOWN)
  - Total Requests
  - Running Pods count
  - App Uptime
  - App Up/Down Over Time (timeseries)
  - Requests Rate Over Time (timeseries)
- **Auto-refresh** every 10 seconds
- **Secure access** via Kubernetes Secrets

### Access Points / ЩҶЩӮШ§Ш· Ш§Щ„ЩҲШөЩҲЩ„

| Service | Port | Access |
|:--------|:----:|:-------|
| Flask App | `30080` | `http://<server-ip>:30080` |
| Prometheus | `30090` | `http://<server-ip>:30090` |
| Grafana | `30030` | `http://<server-ip>:30030` |

## рҹ”’ Security / Ш§Щ„ШЈЩ…Ш§ЩҶ

- вң… **Non-root Docker container** вҖ” Application runs as `appuser`
- вң… **Multi-stage build** вҖ” Minimal attack surface
- вң… **Kubernetes Secrets** вҖ” Sensitive data stored securely
- вң… **GitHub Actions Secrets** вҖ” No credentials in code
- вң… **Tailscale VPN** вҖ” Private network access
- вң… **Custom SSH port** вҖ” Hardened access
- вң… **RBAC** вҖ” Least-privilege Prometheus access
- вң… **Resource limits** вҖ” CPU/Memory quotas on all pods
- вң… **`.gitignore`** вҖ” Protects `terraform.tfvars`, `.env`, `*.tfstate`
- вң… **No secrets in repository** вҖ” All sensitive data via environment/secrets
